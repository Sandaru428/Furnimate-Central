# **App Name**: Furnimate Central

## Core Features:

- User Authentication: Secure email-based login with role-based access control (admin, sales, production, etc.).
- Company Profile Management: Admin interface to update company name, logo, contact info, and branding for system-wide consistency.
- Notification Template Engine: An interface to design customizable email and WhatsApp notification templates triggered by order progress stages, with a preview and testing system.
- Smart Reference Number Generator: Generate unique, date-based reference numbers using a Cloud Function tool to ensure consistency across modules and records. Generative AI evaluates reference strings in an audit to check for compliance.
- Progress Dashboard: Visually track key performance indicators (KPIs) like order pipeline progress, stock alerts, and cash flow.
- Centralized Master Data Management: Manage core data including dynamic product specifications, raw materials, suppliers, and customers.
- Sales Order Flow: Handle the flow from quotation creation and order conversion to dispatch and delivery, updating progress in real-time and triggering automated notifications.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), chosen to reflect trust, reliability, and the technical nature of a management system. Its brightness ensures that it contrasts strongly with the dark background.
- Background color: Very dark blue (#191970) for a sophisticated and modern feel that reduces eye strain.
- Accent color: Electric blue (#7DF9FF) to highlight key actions and elements, providing a modern and technological edge.
- Headline font: 'Space Grotesk' sans-serif for a contemporary, technical feel, especially for headlines; body font: 'Inter', for longer text and comfortable readability.
- Code font: 'Source Code Pro' for any instances of displayed code.
- Use simple, clear icons to represent different modules and actions, maintaining a consistent style.
- A clean, organized layout with clear visual hierarchy and good use of whitespace to avoid feeling cluttered.
